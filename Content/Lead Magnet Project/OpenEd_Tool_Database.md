## Style Tags Legend

Each tool is tagged with compatible homeschool styles:
- **CM** = Charlotte Mason
- **CL** = Classical
- **TR** = Traditional/Structured
- **MO** = Montessori
- **WA** = Waldorf
- **UN** = Unschooling/Interest-Led
- **EC** = Eclectic
- **PB** = Project-Based
- **TF** = Tech-Forward

Age ranges: **E** = Elementary (K-5), **M** = Middle (6-8), **H** = High School (9-12), **A** = All Ages
