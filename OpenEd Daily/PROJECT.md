# OpenEd Daily Newsletter

**Ongoing workflow** - Mon-Thu daily newsletters using Thought-Tool-Trend format.

---

## Quick Start

1. Use skill: `opened-daily-newsletter-writer`
2. Check `CONTENT_QUEUE.md` for upcoming topics
3. Each edition gets its own dated folder

---

## File Structure

```
OpenEd Daily/
├── PROJECT.md              ← You are here
├── CONTENT_QUEUE.md        ← Upcoming topics scheduled by date
└── YYYY-MM-DD - Topic/     ← Individual edition folders
    ├── draft.md            ← Newsletter draft
    ├── sources/            ← Research materials
    └── social/             ← Derivative social posts
```

---

## Workflow

1. **Check queue** - See what's scheduled for today
2. **Run skill** - `/opened-daily-newsletter-writer`
3. **Review draft** - Apply writing rules
4. **Publish** - Send via Beehiiv
5. **Spin spokes** - LinkedIn post, Twitter thread
